### 白板转换器
也可以下载仓库里的转换器，进行导入，导出。
白板转换器会自动识别导入的白板的基本样式，并会根据名称进行图片导入。
在左上角的📂图标点击导入白板。可以在右边预览。
然后下边的图片图标进行导入图片，会白板根据名称进行导入。（如果要导出为html，记得不要导入图片了，图片和网页分开放）
导出后的网页和图片要放在一个文件夹里。


### 功能特性

超大画布（16000x16000）承载丰富内容
完整的游戏时间线记录
各类事件标注
死亡方式等详细分类统计
剧情文本可
基于 Obsidian Canvas 的可视化设计

### 使用方式

下载canvas文件，放进obsidian里使用。

但要注意也要将图片（image文件夹）放进你的仓库里

可在 Obsidian 中直接编辑和扩展内容
