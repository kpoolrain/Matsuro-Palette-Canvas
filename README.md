# 一个基于 Obsidian 的游戏末路调色板时间线白板集合。
### 项目简介

这是一个半成品，未完全，一个demo。只能在obsidian里打开，但后续会添加html的静态形式。

这是一个为游戏设计的综合白板仓库，采用 16000x16000 的超大画布呈现。白板详细记录了游戏的时间线、重要事件、各类成就以及死亡方式等内容，以游戏白板形式进行整体展现。
具有全流程（几乎）

### 功能特性

超大画布（16000x16000）承载丰富内容
完整的游戏时间线记录
各类事件标注
死亡方式等详细分类统计
剧情文本可
基于 Obsidian Canvas 的可视化设计

### 使用方式

下载canvas文件，放进obsidian里使用。

可在 Obsidian 中直接编辑和扩展内容

帮助你自由改进
